## 문항 2: Docker 기본 명령어

### 시험 정보
- 과정: AI 올인원 / AI 네이티브
- 단계: 입학연수
- 난이도: 1
- 권장 시간: 5분
- Pass 기준: 정답 체크리스트 4개 전체 충족 (100점)

### 문제

Docker를 사용하기 위한 **기본 명령어**를 작성하세요.

아래 각 상황에 맞는 **Docker CLI 명령어**를 `answers.py` 파일의 해당 항목에 입력하세요.

| 번호 | 상황 | 답안 키 |
|------|------|---------|
| Q1 | 설치된 Docker의 버전을 확인하고 싶다 | `q1` |
| Q2 | Docker Hub에서 `nginx` 이미지를 다운로드하고 싶다 | `q2` |
| Q3 | 로컬에 저장된 Docker 이미지 목록을 확인하고 싶다 | `q3` |
| Q4 | 중지된 컨테이너를 포함하여 모든 컨테이너 목록을 보고 싶다 | `q4` |

### 제약 사항
- 각 답안은 **한 줄짜리 명령어** 문자열입니다.
- 추가 옵션이나 태그를 넣어도 되지만, 핵심 명령어가 정확해야 합니다.

### 제출 방식
- `answers.py` 파일 1개를 제출합니다.
- 파일 내 `answers` 딕셔너리의 `q1`~`q4` 값을 채우세요.

```python
# answers.py
answers = {
    "q1": "여기에 명령어 입력",
    "q2": "여기에 명령어 입력",
    "q3": "여기에 명령어 입력",
    "q4": "여기에 명령어 입력",
}
```
