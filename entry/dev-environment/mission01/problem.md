## 문항 1: 터미널 기초 - 디렉토리 탐색

### 문제

리눅스/macOS 터미널에서 디렉토리를 탐색하는 기본 명령어를 작성하세요.

#### 작성 예시

> **상황**: 현재 로그인한 사용자 이름을 출력하고 싶다

```python
"q0": "whoami"
```

위와 같이, 각 상황에 맞는 명령어를 `answers.py`에 문자열로 작성하세요.

#### Q1. 현재 위치 확인

현재 작업 디렉토리(Working Directory)의 전체 경로를 출력하세요.

**실행 결과:**
```
/home/user/workspace
```

#### Q2. 디렉토리 생성

현재 위치에 `projects`라는 이름의 새 디렉토리를 생성하세요.

#### Q3. 디렉토리 이동

`projects` 디렉토리 안으로 이동하세요.

#### Q4. 상세 파일 목록

현재 디렉토리의 파일 목록을 출력하세요. `ls` 명령어에 `-l`(상세 정보)과 `-a`(숨김 파일 포함) 옵션을 함께 사용합니다.

**실행 결과:**
```
total 8
drwxr-xr-x  3 user user 4096 Mar  1 10:00 .
drwxr-xr-x  5 user user 4096 Mar  1 09:00 ..
-rw-r--r--  1 user user    0 Mar  1 10:00 .gitignore
```

### 제약 사항
- 각 답안은 한 줄짜리 명령어 문자열입니다.
- 불필요한 옵션 없이 가장 간결한 형태로 작성하세요.

### 제출 방식
`answers.py` 파일의 `q1`~`q4` 값을 채워 제출하세요.

```python
# answers.py
answers = {
    "q1": "",
    "q2": "",
    "q3": "",
    "q4": "",
}
```
