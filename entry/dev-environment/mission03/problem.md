## 문항 3: 터미널 파일 관리

### 문제

터미널에서 파일을 관리하는 기본 명령어를 작성하세요.

#### 작성 예시

> **상황**: `data.csv` 파일의 첫 10줄만 출력하고 싶다

```python
"q0": "head data.csv"
```

위와 같이, 각 상황에 맞는 명령어를 `answers.py`에 문자열로 작성하세요.

#### Q1. 빈 파일 생성

현재 디렉토리에 `README.md`라는 빈 파일을 생성하세요.

#### Q2. 파일 내용 출력

`README.md` 파일의 전체 내용을 터미널에 출력하세요.

#### Q3. 파일 복사

`README.md` 파일을 `docs` 디렉토리 안으로 복사하세요.
명령어 형식: `<복사 명령어> <원본 파일> <대상 디렉토리>`

#### Q4. 파일 삭제

`temp.txt` 파일을 삭제하세요.

### 제약 사항
- 각 답안은 한 줄짜리 명령어 문자열입니다.
- 불필요한 옵션 없이 가장 간결한 형태로 작성하세요.

### 제출 방식
`answers.py` 파일의 `q1`~`q4` 값을 채워 제출하세요.

```python
# answers.py
answers = {
    "q1": "",
    "q2": "",
    "q3": "",
    "q4": "",
}
```
